# Quantum Lab

**SERRE Studio — Interactive Quantum Drug Discovery Simulation**

[▶ Play Online](https://inhaarchive.github.io/quantum-lab-simulation/)

양자 신약 개발 워크플로를 체험하는 교육용 시뮬레이션입니다.

## 주요 흐름

- 연구 환경 선택
- 연구 목표 선택
- Binding Pocket / Candidate
- Active Space
- Hamiltonian
- Qubit Mapping
- Ansatz
- VQE
- Measurement
- Medicinal Chemistry / Redesign
- 연구 보고서 및 프로젝트 보관함

## 안내

이 프로젝트의 수치는 교육용 시뮬레이션을 위한 가상 값이며 실제 약효, 독성, 임상 성공 확률을 의미하지 않습니다.
