# Quantum Lab

양자 신약 개발 워크플로를 체험하는 교육용 시뮬레이션입니다.

## 주요 흐름

- 연구 환경 선택
- 연구 목표 선택
- Binding Pocket / Candidate
- Active Space
- Hamiltonian
- Qubit Mapping
- Ansatz
- VQE
- Measurement
- Medicinal Chemistry / Redesign
- 연구 보고서 및 프로젝트 보관함

## 실행

`index.html`을 브라우저에서 열면 실행됩니다.

## GitHub Pages

이 저장소의 `main` 브랜치 루트를 GitHub Pages 배포 소스로 지정하면 공개 웹사이트로 사용할 수 있습니다.

> 이 프로젝트의 수치는 교육용 시뮬레이션을 위한 가상 값이며 실제 약효, 독성, 임상 성공 확률을 의미하지 않습니다.
